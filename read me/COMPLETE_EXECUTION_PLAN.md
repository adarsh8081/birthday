## 🔧 TROUBLESHOOTING QUICK GUIDE

### "Replit says: Dependencies not found"
```
Fix: npm install framer-motion lucide-react
     Wait 2-3 minutes
     Click Run again
```

### "Website won't deploy"
```
Fix: Make sure app/page.tsx is properly formatted
     Check for syntax errors (red squiggly lines)
     Try refreshing browser
     Clear cache (Ctrl+Shift+Delete)
```

### "Website looks weird on iPhone"
```
Fix: 1. Hard refresh (Cmd+Shift+R on Mac)
     2. Clear Safari cache
     3. Test in another browser (Chrome)
     4. Verify viewport meta tags
```

### "Animations are laggy"
```
Fix: 1. Deploy to Vercel (faster than Replit)
     2. Reduce particles: [...Array(20)] → [...Array(10)]
     3. Disable on slow 4G connections
```

### "Text is too small on iPhone"
```
Fix: It's already optimized (responsive)
     If still small: Settings → Accessibility → Text Size
     Should increase automatically
```

### "Page takes 5+ seconds to load"
```
Fix: 1. Check internet speed
     2. Use Vercel instead of Replit
     3. Reduce animations/particles
     4. Test on different network
```
